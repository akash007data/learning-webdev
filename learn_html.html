HTML Concepts Revision

<!DOCTYPE html>
Declares the document type and version of HTML (HTML5). It helps browsers render the page correctly.
<html>
The root element of an HTML page. All content goes inside this tag.
<head>
Contains meta-information (not shown on the page), like title, character encoding, and links to CSS/JS.
<meta charset="UTF-8">
Sets the character encoding to UTF-8, supporting most languages and symbols.
<title>
Sets the title of the page (shown in browser tab).
<style>
Used to write CSS (Cascading Style Sheets) for styling HTML elements.
